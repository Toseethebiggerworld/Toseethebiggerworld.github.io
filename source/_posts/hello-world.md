博客已经初始化完成，这是一篇默认文章，你可以把它删掉，或者直接改成你的第一篇正式内容。

## 现在可以做什么

1. 修改站点名称、作者和网址
2. 本地启动预览
3. 创建新的 Markdown 文章
4. 推送到 GitHub Pages 自动发布

## 常用命令

```bash
npx hexo new "我的新文章"
npx hexo server
npx hexo generate
```

## 图片怎么放

这个项目已经开启了文章资源文件夹。以后创建文章时，可以给每篇文章配自己的图片目录，管理起来会更清晰。
